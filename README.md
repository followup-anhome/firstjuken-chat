# firstjuken-chat

ファースト住建 現場チャット
3言語AI自動翻訳（日本語・タガリッシュ・ベトナム語）

## 開発元
フォローアップ株式会社

## 技術スタック
- Next.js 15.2.6
- Supabase（リアルタイムチャット）
- Claude Haiku API（claude-haiku-4-5-20251001）

## 起動方法
```
npm install
npm run dev
```

## 環境変数（.env.local）
- ANTHROPIC_API_KEY
- ANTHROPIC_MODEL
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
